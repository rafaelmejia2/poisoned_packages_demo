# poisoned_packages_demo

This is a simple web application used to simulate downloading an unknown dependency through npm.

Quick Start

# Prerequisite

- Docker Desktop
- Git

1. Clone repo
2. cd poisoned_packages_demo
3. Run the stack with `docker compose up -d`
4. Wait until docker desktop has log of localhost:3000 being started.
5. [Visit:](http://localhost:3000)
6. Stop everything with `docker compose down`
